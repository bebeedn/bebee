## Как это работает

1. Пользователь заполняет форму на любой странице сайта (Садочок, Школа, Додаткові заняття)
2. При отправке формы данные отправляются на `/api/telegram`
3. API отправляет сообщение в ваш Telegram бот
4. Вы получаете уведомление с данными клиента

## Как создать нового бота (если нужно)

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям (придумайте имя и username для бота)
4. Получите токен бота (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. Узнайте свой Chat ID:
   - Найдите бота **@userinfobot** в Telegram
   - Отправьте ему любое сообщение
   - Он пришлет вам ваш Chat ID (формат: `123456789`)
6. Замените токен и Chat ID в коде или в `.env.local`

## Тестирование

1. Запустите сайт локально: `npm run dev`
2. Откройте страницу Садочок: `http://localhost:3000/kindergarten`
3. Нажмите "Записатися в садочок"
4. Заполните форму и отправьте
5. Проверьте Telegram - должно прийти сообщение

## Деплой на Vercel

Если вы используете Vercel, добавьте переменные окружения в настройках проекта:

1. Откройте проект на Vercel
2. Settings → Environment Variables
3. Добавьте:
   - `TELEGRAM_BOT_TOKEN` = ваш токен
   - `TELEGRAM_CHAT_ID` = ваш chat ID
4. Redeploy проекта

## Поддержка

Если что-то не работает:
1. Проверьте, что бот запущен (отправьте ему `/start` в Telegram)
2. Проверьте правильность токена и Chat ID
3. Проверьте консоль браузера на наличие ошибок (F12 → Console)
4. Проверьте логи сервера
